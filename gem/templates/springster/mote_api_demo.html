We're using the button element in a few different ways.

The button's default data. Note children is a dictionary / associative array and not a list, because we need to be able to perform deep merges: 
        {
            "tag": "button",
             "children": {
                "text": {
                    "id": "self.browser.atoms.text",
                    "value": "Button text"
                }
            }
        }

Just change the text. Not we only have to provide a subset of the data declared in data.yaml. Mote will deepmerge our data with the sample data.
